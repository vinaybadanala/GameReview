This is my Game Review Website developed to showcase my skills in frontend development and UI design. The project focuses on presenting game reviews, ratings, and details in a clean, modern, and user-friendly interface. It demonstrates my ability to design responsive web pages and organize content effectively for a better user experience. The website includes structured sections for showcasing games, making it easy for users to explore and understand different game reviews.

This project is built using modern web technologies including HTML5, CSS3, and JavaScript. It highlights my understanding of responsive design, layout structuring, and styling techniques. Tools like Git and GitHub were used for version control and deployment through GitHub Pages. The main goal of this project is to demonstrate my frontend development skills, creativity, and ability to build real-world web applications.
